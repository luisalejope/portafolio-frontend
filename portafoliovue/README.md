## Project Setup
With npm
```sh
npm install
```
with yarn
```sh
yarn
```

### Compile and Hot-Reload for Development
With npm
```sh
npm run dev
```
with yarn
```sh
yarn dev
```

### Compile and Minify for Production
With npm
```sh
npm run build
```
with yarn
```sh
yarn build
```
